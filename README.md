# sai-na/homebrew-tap

Homebrew formulae for [sai-na](https://github.com/sai-na) tools.

```bash
brew install sai-na/tap/pinfolder
pinfolder-setup   # copies the app to /Applications, registers the Quick Actions
```

| Formula | Description |
|---|---|
| [`pinfolder`](Formula/pinfolder.rb) | [PinFolder](https://github.com/sai-na/PinFolder) — pin files & folders on macOS: menu bar, top of folder, or Finder sidebar |
